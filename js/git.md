# git 使用 
```javascript
git clone
git status
git branch
git diff
git add
git commit -m "msg"
git log
git show (git log id)
git checkout
git push origin master
git pull origin master
// 多人开发
// 创建新分支
git checkout -b feature
git checkout feature
git push origin feature
git checkout master
// 拉下所有分支
git fetch
git checkout feature
// 更新分支最新内容
git pull origin feature
git checkout master
// 合并分支
git merge feature
git push origin master
// 移除修改并暂存
git stash
```
