# 值类型和引用类型
## 值类型
> 数据存储在栈内存
```javascript
let a = 100
let b = a
a = 200
console.log(b) // 100
```
### 常见值类型
```javascript
let a // undefined
let b = 'b'
let c = 100
let d = true
let e = Symbol('e')
```

---
## 引用类型
> 数据存储在堆内存  
> 存储的是实际数据的地址
```javascript
let a = {age: 20}
let b = a
// b 和 a 存储的地址相同
// 改变 b 所指向的实际数据会同时改变 a 所指向的数据
b.age = 21
console.log(a.age) // 21
```
### 常见引用类型
```javascript
const obj = {x: 100}
const arr = [1, 2, 3]
const n = null // 空指针
const f = function() {}
```
